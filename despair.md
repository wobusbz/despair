###项目结构目录如下

```buildoutcfg
├─app                   应用目录
│  │─index    
│  │  ├─controller      控制器目录 
│  │  ├─model           模型目录 
│  ├─middleware         中间件目录
│  ├─routers            路由文件目录
├─common      			模块函数文件(小函数)
├─conf                  配置文件目录
├─Db                    数据库目录
├─logic                 逻辑层
```
